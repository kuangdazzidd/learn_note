安装synergy软件
###############


**1.安装**
    直接通过apt安装

   >>> sudo apt-get install synergy

**2.安装gui**    
    >>> sudo apt-get install quicksynergy


**3.windows端**
    电脑端直接安装synergy软件充当服务器
